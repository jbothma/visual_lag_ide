Help and Documentation

  • Editor overview

    PEAL has four tabs. The LAG editor is the default active tab. Other tabs can be activated by left-clicking on them.

    The tab and its border change colour to indicate which tab is active.



  • Quick tips
    • Inserting visual elements

      Visual elements can be inserting into the appropriate containers by right-clicking inside the container, then selecting the desired visual element from the Insert menu.

      See also:
      Visual element features

    • Deleting visual elements

      Visual elements can be removed from the strategy by right-clicking on the element, then clicking the Delete option.

    • Moving visual elements

      Visual elements can be moved to other containers or reordered within a list by dragging the element with the mouse.

      Click and hold on the element to be dragged, then move the mouse to the desired position.
