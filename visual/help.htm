Visual LAG Editor Help

  • Delete Tool
    • Click on 'Delete Tool', then on a visual element in the editor to delete it.
  • LAG Attribute
    • The Attribute menu lets you choose each layer of a LAG attribute to insert.
    • Text input fields in the menu allow you to insert custom attribute names.
    • Attributes can be dragged around the editor window once inserted.
  • Connector
    • Click the Connector menu item to insert a new connector.
    • The black squares at each end of the connector are the handles
    • Connectors can be moved around the screen by dragging their handles around.
    • Drop connector handles on Connectable Box Connection Points to connect the Connector to the Connectable Box
  • Connectable Box
    • Click the Connectable menu item to insert a new Connectable Box.
    • The green squares above and below the box are Connection Points.
    • See Connector on how to connect Connectable Boxes.
